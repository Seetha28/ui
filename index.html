Modern Portfolio Pro by StudioPress

// Updated On: May 8, 2016 Your comments? // Cats: Themes Featured, Themes Premium // Tags: 2 column, 2013 release, Google web fonts, HTML5, mobile responsive design, mobile specific menu, page templates, StudioPress, widgetized home page

The Modern Portfolio Pro theme from StudioPress is an HTML5-compliant, mobile responsive minimalist design featuring a widgetized home page. It’s a refresh of the Modern Portfolio theme with a few changes:

  • support for HTML5 markup
  • a new after post widget area
  • support for multiple color styles
  • a setting to set the single character used in the site header
  • dropping support for custom backgrounds

Introduced on 12/10/13, the Modern Portfolio Pro is a 2 column theme with 3 layout options. Three of the 6 standard Genesis layout options have been removed to simplify content creation. The theme includes a landing page template. Beyond the WordPress standards of custom menus and threaded comments, the Modern Portfolio Pro theme also supports custom headers, multiple color styles (blue, orange, purple, red, and green – the default), and Google web fonts (specifically Lato and Merriweather). The selected fonts and weights add about 140 Kb to the site’s page weight, which is a big addition for typography.
